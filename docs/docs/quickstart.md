## Quickstart

#### Get started with Docmach in a few simple steps.
---

1. **Install Docmach**
   ```bash
   npm install -g docmach
   ```
2. **Create a new directory**
   ```bash
   mkdir my-docs
   cd my-docs
   ```
3. **Start the development server**
   ```bash
   docmach
   ```
5. **Open your browser and navigate to `http://localhost:4000`**
